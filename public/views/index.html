Manual

Type route to start.


Route command comes with different options:

-all: get all possible routes of two nodes.

-c (cheapest): get the cheapest possible route of two nodes.

-s (set): get Delivery cost of the given delivery route.


Examples:

route -all a-b

route -c a-b

route -s a-b-e


To run the command, press Enter

To clear the screen, press ctrl + l
